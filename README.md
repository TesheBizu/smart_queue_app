# Smart Queue App

A Flutter + Firebase mobile application that manages queues for clinics, banks, and service providers.

## Features
- Multi-service queue system
- Admin queue control
- Provider manual claim system
- Real-time updates using Firestore

## Tech Stack
- Flutter (Dart)
- Firebase (Auth, Firestore, Cloud Functions)
- Riverpod

## System Design
- Shared queue per service
- Multiple providers per service
- Ticket lifecycle management
